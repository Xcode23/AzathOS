# AzathOS

Hobby kernel made to learn how OS kernels are implemented.

## Roadmap

* Framebuffer based Text UI :heavy_check_mark:
* Keyboard Input :heavy_check_mark:
* Paging
* Memory management
* Klib data-structures: string, map, vector...
* Serial Output Logging
* Process management
* Filesystem
* Uefi support
* x86_64 support
* Arm support
